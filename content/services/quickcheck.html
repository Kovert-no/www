Frequently Asked Questions

What is the difference between a penetration test and a vulnerability scan?

A vulnerability scan is an automated test, while a penetration test is done by a person who examines the systems using manual techniques. An automatic scan only identifies the vulnerabilities, while a penetration test provides a more informative picture of how the vulnerability occurs, and to what extent the vulnerability can be exploited (simply a simulated hacker attack).
A vulnerability scan can also contain false positives, and will not be able to identify ALL vulnerabilities in the system. A pentest will be able to identify more vulnerabilities than a scan, as well as verify any false positives.
A vulnerability scan is in most cases the first step in a more comprehensive penetration test. Therefore, such a scan provides a good basis for further security testing as well.
